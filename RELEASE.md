# Mocpot v1.1.0 发布说明

## 下载

[最新版本](https://github.com/LINEXIANSEN/Mocpot/releases/latest)

## 新增功能

### 画中画 (Picture-in-Picture) ⭐
- macOS 原生画中画支持
- 退出全屏时自动进入画中画
- 最小化时自动进入画中画
- 快捷键 `P` 快速切换

### 主题切换 ⭐
- 跟随系统主题
- 浅色模式
- 深色模式
- 设置中可切换

## 更新内容

### 修复
- 修复 VR 360° 全景视频切换卡住问题
- 修复进度条拖动回弹问题
- 修复全屏按钮无效问题
- 修复空格键播放/暂停问题
- 修复打开文件不自动播放问题
- 修复视频画面纯灰问题
- 修复播放列表排序无效问题

### 新增
- VR 视频单击播放/暂停
- 文件夹导入功能（自动扫描视频）
- 文件夹导入排序（名称/时间/自然顺序）
- 播放列表排序功能
- 随机播放按钮
- VR 操作提示 2 秒自动消失
- 播放速度 0.25x - 4x
- A-B 循环标记显示
- 快速设置面板
- 8 个设置标签页
- 完整快捷键支持
- 画中画功能
- 主题切换功能

### 优化
- 清理未使用的代码文件
- 优化项目结构
- 改进手势识别

## 快捷键

| 按键 | 功能 |
|------|------|
| Space | 播放/暂停 |
| F | 全屏 |
| P | 画中画 |
| ← / → | 快退/快进 10 秒 |
| ⌘ + ← / → | 快退/快进 5 秒 |
| ⌘ + O | 打开文件 |
| ⌘ + ⇧ + O | 导入文件夹 |
| ⌘ + L | 播放列表 |
| ⌘ + S | 截图 |
| ⌘ + D | 3D 模式 |
| ⌘ + V | VR 模式 |
| A | 设置 A 点 |
| B | 设置 B 点 |

## 系统要求

- macOS 13.0 (Ventura) 或更高版本
- 支持 ARM64 (Apple Silicon) 和 x86_64 (Intel)

## 安装

1. 下载 `Mocpot-1.1.0.dmg`
2. 双击打开 DMG
3. 将 Mocpot 拖入 Applications 文件夹
4. 首次打开：右键 → 打开
